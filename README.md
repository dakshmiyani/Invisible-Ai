# Invisible AI: The Seamless Writing Assistant

Invisible AI is a high-fidelity, Google Docs-style Chrome extension designed for professional writing and AI-assisted content creation. It provides a pixel-perfect document editing environment with a powerful, "invisible" AI assistant that streams content directly into your workspace.

## 🌟 Overview

Conventional AI assistants often rely on clunky sidebars, modals, or explicit "AI" tags that break your writing flow. **Invisible AI** changes this by making the artificial intelligence a natural extension of your own typing. It maintains the look and feel of a native Google Docs editor while providing state-of-the-art LLM capabilities.

---

## ✨ Key Features in Detail

### 1. Professional WYSIWYG Editor
- **Rich Text Formatting**: Full support for Bold, Italic, Underline, and Text Color via a native-feel toolbar.
- **Dynamic Typography**:
  - Incremental Font Size controls (pinned to standard Google Docs increments).
  - Cycle through professional font families (Arial, Georgia, Courier New, etc.).
- **Document History**: Built-in Undo and Redo functionality integrated with the browser's command history.
- **Standard Navigation**: Intuitive "File", "Edit", "View", "Insert", "Format", and "Tools" menus with functional dropdowns.

### 2. The "Invisible" AI Assistant
- **Direct Streaming Integration**: Content generated by the AI is streamed token-by-token directly into the editor. No intermediate popups or "copy-paste" required.
- **Context-Aware Memory**: The assistant analyzes the last **three blocks** of your document to ensure continuity in style, tone, and subject matter.
- **Intent Detection**: The system automatically detects whether you are writing a standard report (prose) or a technical script (code) and adjusts its output format to match.
- **Selection-Based Improving**: Highlight any text and click "Ask" to have the AI rewrite, polish, or expand upon specifically what you've selected.

### 3. Professional Export System
- **Word (.doc) Compatibility**: Export your documents in a format strictly optimized for Microsoft Word. 
- **MSO-Compliant Headers**: Uses specialized Office XML/HTML headers to ensure your document opens in Word with all formatting (headings, code blocks, bold/italic) intact and without "Unreadable Content" errors.

## 🛠️ Technical Architecture

- **Manifest V3**: Built on the latest Chrome Extension standards for security and performance.
- **Groq API Integration**: Utilizes the ultra-fast `llama-3.3-70b-versatile` model for near-instant response streaming.
- **Zero-CDN Policy**: All libraries (like `marked.js` for markdown parsing) are bundled locally within the extension to ensure stability and privacy.
- **State Management**: Implements a redundant auto-save system that syncs the document state to `chrome.storage.local` every 30 seconds.
- **Security**: Your API Key is stored securely in local extension storage and is never transmitted anywhere except directly to the Groq API.

## 🚀 Installation & Setup

### Prerequisites
- A **Groq API Key**. Generate one for free at [console.groq.com](https://console.groq.com/keys).

### Step-by-Step Installation
1. Download or clone this repository to your local machine.
2. Open Google Chrome and go to `chrome://extensions/`.
3. Toggle on **Developer mode** in the top-right corner.
4. Click **Load unpacked** and select the extension folder containing `manifest.json`.
5. Pin **Invisible AI** to your toolbar for easy access.

### First-Run Configuration
- Upon opening the extension for the first time, you will be prompted to enter your Groq API Key. This is required to activate the "Ask" features.

## ⌨️ Shortcuts & Tips

- **Magic Write**: Press `Ctrl + Enter` (or `Cmd + Enter` on Mac) to trigger the AI assistant at any time.
- **Smart Improve**: If you want to refine a specific sentence, **highlight it** first and then click **Ask**. The AI will focus solely on improving that selection.
- **Placeholder**: The "Start writing..." hint disappears automatically the moment you focus on the editor, allowing for a clean start.
- **Fullscreen**: Use the **View > Fullscreen** menu option for a completely distraction-free writing environment.

## 📂 Project Structure

- `manifest.json`: Extension identity and permission set.
- `popup.html`: The core HTML5 structure of the editor.
- `script.js`: The "brain" of the extension—handles editor commands, storage, and API communication.
- `styles.css`: The UI engine—responsible for the high-fidelity Google Docs aesthetic and layout constraints.
- `marked.min.js`: A specialized markdown parser used to render AI responses with code highlighting support.
